Одесский кружок "Медиа-ключ" (последователи ММК)
выделяют следующие виды деятельности в бизнесе

| 1   |                               | Наезд             | 123 |
| --- | ----------------------------- | ----------------- | --- |
| 2   | Предметная                    |                   | 234 |
| 3   |                               | Борьба за ресурсы | 345 |
| 4   | Игровая-коммуникативная       |                   | 456 |
| 5   |                               | Манипуляция       | 567 |
| 6   | Рефлексивная-смыслообразующая |                   | 678 |
По номерам слева это
2-4-6 конструктивные деятельности
1-3-5 деструктивные деятельности

В правом столбце - тройки уровней ТУАИ, которые схватывают содержание этих видов деятельности
