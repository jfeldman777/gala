На основе элементарного конфликта возможна следующая модель.

Игроки ходят по очереди. У каждого есть свой набор уровней (4 числа из множества 12345678)

Всего надо сыграть 8 партий, каждый должен начать одну партию с каждого числа.

То есть игроки А и В играют 8 партий, каждый начинает 4 партии, по одной с каждого числа.

Ход можно сделать только из своего набора чисел.

Ход выбирается таким образом, чтобы это число и последний ход противника составляли наилучший для тебя результат.

Игра заканчивается либо ситуацией ВОЙНА либо повторениям ходов

Если есть несколько возможных ходов с одинаковым результатом выбирается наибольшее число

После 8 партий, считаем сколько из них закончилось войной

Применение.

Если А имеет право выбрать партнера из Б и В, он считает профиль свой и партнера, проводит маневры и вычисляет количество войн в каждой паре (АБ) и (АВ).

Если с партнером есть хотя бы одна война и можно его не брать, то лучше не брать.
Если брать обязательно, нужно брать того с кем меньше войн
