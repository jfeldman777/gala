At university we were taught automata theory.
It turns out automata have complexity classes.

The simplest automaton has no memory.
A doorbell, for example.
Press it — it rings.

The next automaton up in complexity has finite memory.
A washing machine, for example.
You set a mode — and it runs according to that mode.

Next up in complexity is an automaton with infinite memory.
Your computer, for example. If needed, you can buy more memory.

Finally, the fourth complexity class includes your computer connected to a network.
Automata of this class are able to modify their own program.
For example, your computer can download updates for all its programs on its own.

So, which class are you in?
